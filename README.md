Android Calculator App

Project Description

This project is a simple Android Calculator Application developed using Java in Android Studio. The
application allows users to perform basic arithmetic operations through a clean and user-friendly
interface.

Features

• Addition
• Subtraction
• Multiplication
• Division
• Clear input and display result
• Simple and responsive user interface

Technologies Used

• Java
• Android Studio
• XML Layouts
• Gradle Build System

Installation and Setup

1. Clone the repository from GitHub.
2. Open the project using Android Studio.
3. Allow Gradle to
sync the project.
4. Run the application on an emulator or Android device.
   
Purpose of the Project

The main purpose of this project is to understand Android application development using Java,
implement event handling, design user interfaces using XML, and apply basic arithmetic logic within
an Android environment.
